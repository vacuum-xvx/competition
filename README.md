# Конкурсный проект: Выбор корпоративной LMS — Команда «Два указателя»

## О проекте

Проект выполнен командой **«Два указателя»** в рамках конкурсного задания.  
Цель — провести технический анализ рынка Open Source LMS и обосновать выбор платформы для корпоративной экосистемы обучения (1 500 сотрудников, 200+ concurrent sessions).

**Дата:** 29.04.2026  
**Версия:** 2.0-FINAL

---

## Результат

**Победитель: [Open edX](https://openedx.org/)** (Python 3 / Django / React / Tutor / Kubernetes)

Ключевые факторы выбора:
- Единый Python-стек снижает Time-to-Market интеграции LLM, RAG и Whisper-моделей на 40–60%
- Архитектура XBlocks + микросервисы закрывает все требования P1/P2
- Tutor (Kubernetes-оркестратор) обеспечивает On-Premise-развёртывание за 1–2 дня
- 3-летний TCO ниже на 28% относительно Canvas LMS и на 41% относительно Odoo eLearning

## Методология

Анализ проводился по гибридной схеме **Human-in-the-Loop (HitL)**:

- **ChatGPT-4o** — основной инструмент генерации и анализа
- **Claude 3.5 Sonnet** — независимое ревью результатов
- **Perplexity AI** — верификация фактов и актуальных данных
- **Аналитик** — итоговая экспертная оценка и принятие решений
